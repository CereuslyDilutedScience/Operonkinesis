## Overview
The concept of OperonKinesis is to create an interactive UI simulator to visualize and explain how operons regulate gene expression.

## Architecture:
*  Modular desktop app
*  Event-driven simulator generates output only when the user clicks a generate button.

## Tech Stack
*	Language: Java
*	Build: gradle
*	UI: JavaFX
*	Serialization: Jackson
*	Logging: SLF4J + Logback
*	Testing: JUnit 5, TestFX
